# FNMA_class_project

## Before you run
Make sure you have set up everything  
  * Such as postgresql  
  * Run npm install  
  * config.json values are not default  


### Running
Verify the postgreSQL database is running with:  
  `$ sudo service postgresql start`  

In the FNMA_class_project directory, run:  
 `$ .start_server.sh`
